# 💰 Personal Expense Tracker
A clean, responsive, and user-friendly web application to manage personal finances. Track your daily income and expenses effortlessly, with real-time balance updates and local storage support so your data is always safe.

## ✨ Features
* **Real-Time Calculation:** Automatically updates your total balance, total income, and total expenses the moment you add a new transaction.
* **Persistent Storage:** Uses the browser's `localStorage` so your transaction history is saved even if you close or refresh the tab.
* **Custom Currency Formatting:** Automatically formats all numbers into Bangladeshi Taka (BDT / ৳) using standard currency conventions.
* **Modern, Responsive UI:** Features a sleek custom color theme, smooth CSS animations, hover effects, and a fully responsive grid layout that looks great on both desktop and mobile devices.
* **Quick Management:** Easily add descriptive transactions and remove individual items with a single click.

## 🛠️ Technologies Used
* **HTML5:** Semantic markup and structure.
* **CSS3:** Custom styling, Flexbox/Grid layouts, custom fonts (Lato & League Gothic), and media queries for mobile responsiveness.
* **Vanilla JavaScript (ES6+):** DOM manipulation, event handling, array filtering/reducing, and `localStorage` integration.

## 🚀 How to Run Locally
Since this project uses pure Vanilla JavaScript, HTML, and CSS, there is no need for `npm` or complex server setups.
